# FindMyVibe — Frontend

React 기반 성향 분석 + 추천 웹 SPA

> Phase 3에서 구현 예정

## 기술 스택 (계획)

| 항목 | 기술 |
|------|------|
| Framework | React 19 |
| Language | TypeScript |
| Build | Vite |
| Styling | Tailwind CSS 4 |
| Data Fetching | TanStack Query |
| Form | React Hook Form |

## 주요 화면 (계획)

1. **질문 화면** — 기본 질문 7개 + AI 꼬리질문 순차 표시
2. **성향 프로필** — 타입 라벨 + 레이더 차트 + 키워드
3. **추천 결과** — 매칭 점수 + 원데이클래스 카드 목록

## 로컬 실행 (구현 후)

```bash
npm install
npm run dev
# http://localhost:5173
```
