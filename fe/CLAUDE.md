# CLAUDE.md — FindMyVibe Frontend

> Phase 3에서 구현 예정

## 기술 스택
- React 19 + TypeScript + Vite
- Tailwind CSS 4
- TanStack Query + React Hook Form

## 명령어 (구현 후)
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm test`

## API 연동
- Base URL: `http://localhost:8080` (로컬) / `https://api.findmyvibe.com` (운영)
- 모든 API는 `/api/v1/` prefix
